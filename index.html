The Colours of Monet

After a visit to the MOMA I became a little fixated with how one might automatically detect colour palettes from images. I used Python to scrape a large collection of Monet paintings and implemented k-nearest-neighbours with some colour correction using Python Imaging Library to extract primary colour groups. I completed the project by creating a data visualization using D3.js.
